# Website Layout

## Overall Visual Style
- Single-page portfolio with a centered max-width shell.
- Electric Sunstone theme: warm coral-orange accents, sunstone gold highlights, soft cream/light surfaces in light mode, and deep brown/orange surfaces in dark mode.
- Light mode is the default. A dark mode toggle is available in the header.
- Cards and panels use compact 8px-radius corners, thin warm borders, and restrained shadows.
- Layout is responsive: desktop uses multi-column grids; tablet/mobile stacks sections vertically.
